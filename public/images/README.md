# Инструкции за снимки на студиата

## Структура на папките

Всяко студио има собствена папка със снимки:

```
public/images/
├── apartment1/     (Апартамент №1)
│   ├── main.jpg              - Главна снимка (спираловидно стълбище с кухня)
│   ├── spiral-kitchen.jpg    - Стълбище и кухня
│   ├── living-room.jpg       - Всекидневна
│   ├── bedroom1.jpg          - Спалня 1 (двойно легло)
│   ├── bedroom2.jpg          - Спалня 2 (две единични легла)
│   ├── bedroom3.jpg          - Спалня 3 (три легла - горен етаж)
│   └── bathroom.jpg          - Баня
├── studio2/        (Студио №2)
│   └── main.jpg
├── studio3/        (Студио №3)
│   └── main.jpg
├── apartment4/     (Апартамент №4)
│   └── main.jpg
├── studio5/        (Студио №5)
│   └── main.jpg
├── studio6/        (Студио №6)
│   └── main.jpg
└── guesthouse/     (Olive Garden Guesthouse)
    └── main.jpg
```

## Как да добавите снимки

1. **Запазете снимките** от оригиналния сайт или вашата библиотека
2. **Преименувайте ги** според структурата по-горе
3. **Копирайте ги** в съответните папки
4. **Рестартирайте** development сървъра (`npm run dev`)

## Забележки

- Използвайте **JPG формат** за по-малък размер
- Препоръчителна резолюция: **1200x800px** или по-висока
- Име на файловете: **малки букви, с тире** (например: `spiral-kitchen.jpg`)
- **main.jpg** е задължителна за всяко студио

## Допълнителни снимки

Ако искате да добавите повече снимки за дадено студио:
1. Копирайте снимките в папката на студиото
2. Актуализирайте `images` масива в базата данни (чрез Supabase Dashboard или SQL)
